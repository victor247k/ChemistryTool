Quick Start Guide

My API is incredibly easy to use, check out the code samples bellow to get started:

Fetch an Element

All you need to do is send a request to:

  • https://[thiswebsite]/api/all/
  • https://[thiswebsite]/api/elements?symbol=[symbol]

    • where:

  • [symbol] - is the symbol of the element you want ot get information on.

Sample Requests

https://[thiswebsite]/api/all/ - Get info on all elements

https://[thiswebsite]/api/elements?symbol=H&symbol=O - Get info on the elements Hydrogen and Oxygen
